# RLDDM

## Overview

`RLDDM` is an R package that provides a computational framework for modeling decision-making processes using a combination of reinforcement learning (RL) and drift-diffusion model (DDM) using Frequitist methods. The details of the RLDDM can be found in found in [Pedersen et al., (2017)](https://doi.org/10.3758/s13423-016-1199-y) (for Bayesian) and [Bian et al., (2025)](https://doi.org/10.1214/25-AOAS2112) (for Frequistist) respectively.

## Installation

To install `RLDDM`, use the following command:

``` r
remotes::install_github("https://github.com/krisyuanbian/RLDDM")
```
